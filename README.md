## üöÄ Quick Start

### Local Development

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No build process required.

### GitHub Pages Deployment

1. Create a new GitHub repository
2. Upload all files (`index.html`, `styles.css`, `script.js`)
3. Go to repository Settings ‚Üí Pages
4. Select the main branch as the source
5. Your site will be published at `https://yourusername.github.io/repository-name`

## üé® Customization

### Update Personal Information

1. **Hero Section**: Edit name, title, and tagline in `index.html` (lines 40-60)
2. **About Me**: Update bio content (lines 100-150)
3. **Skills**: Modify skill tags in each category (lines 180-250)
4. **Experience**: Update job details and descriptions (lines 280-350)
5. **Projects**: Replace project cards with your own (lines 380-480)
6. **Contact**: Add your email and social media links (lines 550-600)

### Color Scheme

Edit CSS variables in `styles.css` (lines 10-30):

```css
:root {
    --primary-color: #667eea;
    --accent-color: #4facfe;
    /* ... more colors */
}
```

### Add Your Projects

Replace placeholder project cards in `index.html`:

```html
<div class="project-card">
    <div class="project-image">
        <!-- Add background image or gradient -->
    </div>
    <div class="project-content">
        <h3 class="project-title">Your Project Name</h3>
        <p class="project-description">Description...</p>
        <div class="project-tech">
            <span class="tech-tag">Technology</span>
        </div>
    </div>
</div>
```
